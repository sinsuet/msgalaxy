# Docs Archive

This directory stores ADRs and reports that no longer describe the active
scenario-runtime mainline.

Current active documentation lives in:

- `docs/adr/`
- `docs/reports/`
- `HANDOFF.md`
- `README.md`

Archive layout:

- `docs/archive/adr/` for retired ADRs
- `docs/archive/reports/` for retired reports

When a document stops reflecting the current mainline, move it here instead of
keeping it in the active docs surface.
